# Code Of Conduct

## Purpose

KE8YGW Logger should be a practical and respectful project for amateur radio
operators, developers, testers, and maintainers. Participation requires
professional conduct, clear technical communication, and respect for other
contributors' time and safety.

## Expected Behavior

- Be respectful and constructive.
- Keep discussion focused on the project and its users.
- Prefer specific technical arguments over personal criticism.
- Accept maintainer decisions once they are made.
- Protect user privacy, credentials, station data, and operational details.
- Report security concerns through the private process in [SECURITY.md](SECURITY.md).

## Unacceptable Behavior

- Harassment, threats, discrimination, or personal attacks.
- Publishing another person's private information without permission.
- Sharing credentials, private keys, LoTW certificate material, production data,
  or unredacted diagnostics.
- Publicly posting exploit details before a coordinated security response.
- Repeatedly derailing issues or pull requests after maintainers have asked for a
  change in direction.

## Enforcement

Maintainers may edit or hide comments, close issues, block participants, reject
contributions, or limit repository access when behavior harms the project or its
users. Enforcement decisions should be proportionate, documented when practical,
and focused on keeping the project safe and useful.
